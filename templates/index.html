About This Platform

This platform is designed to help you learn about web application security through practical examples. It contains several intentionally vulnerable features that simulate real-world security issues.

Your challenge is to identify and exploit these vulnerabilities to find the hidden flags.

Remember, the skills you learn here should only be applied to systems you have permission to test!
